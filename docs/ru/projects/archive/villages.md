# Деревни

#### Проект подразумевал собой строительство системы деревень на Центральном острове и их объединение в единый "деревенский квартал". В рамках проекта были построены 3 деревни: "Первая", "Вторая", "Третья".

## Первая деревня

#### Основное здание состоит из кубических модулей, расположенных на сваях. Строительство было начато 09.07.2024, именно в этот день были открыты 4 модуля для жителей. В будущем деревня была реконструирована с внедрением дополнительного этажа для жителей.
