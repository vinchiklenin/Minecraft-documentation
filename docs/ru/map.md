# Карта
Центром карты нашего проекта является остров "Центральный", на котором расположены нулевые координаты 0, 0, 0. Карта остается неизменной с момента открытия. Выживание проходит без "вайпов" и откатов, однако из-за того что сервер переходил на иные версии, на отдаленный координатах можно заметить новые структуры, блоки и мобов.
{% note tip %}
В целях экономии дискового пространства не рекомендуется бесцельно длительно перемещаться в сторону закрытого направления (где отсутствует транспортное сообщение + нет интересного объекта далее)
{% endnote %}

## Транспорт
{% note tip %}
Настоятельно рекомендуется использовать транспорт при путешествиях
{% endnote %}
Поскольку "Центральный" остров расположен посреди океана, при посещении нулей придется задействовать транспорт. При использовании транспорта необходимо соблюдать правила при эксплуатации каждого из видов. Познакомимся с возможными способами путешествий ниже:
Самые популярные:
* Железные дороги (вагонетки на рельсах)
- Путешествия по железным дорогам возможны по всем территориям где проложены железнодорожные пути и сделаны остановочные пункты (рекомендуется ознакомиться с транспортной картой)
* Метро (лодки на льду)
- Путешествия под землей по синему льду с высокой скоростью возможны по прорытым тоннелям и сделанным путям (рекомендуется ознакомиться с транспортной картой)
Иные способы путешествия:
* Корабли (лодки в воде)
- Путешествия возможны только по воедоемам (рекомендуется ознакомиться с обычной картой)
* Ездовые мобы (Лошадь, осел, лама, лавамерка, свинья)
- Путешествия возможны по биомам, не мешая железнодорожному транспорту
