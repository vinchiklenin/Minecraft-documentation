# Устав ТСЖ Vermeer Building

## 1. Основные положения
1.1 Адресом здания "Vermeer Building" закрепляется ул. Кудринская 2, Центрального Архипилага
1.2 Положение Устава ТСЖ распространяется на каждого проживающего(резидента) и посещающего(гостя) "Vermeer Building"
1.3 Нарушение правил влечет за собой наложение санкций в соответствии с правилами
1.4 Наименования "Vermeer Building", Высотка, "Vermeer Tower" равнозначны
1.5

## 2. Жилищные правила
2.1 Запрещается внесение изменений в несущих конструкциях высотки (стены, фасад и т.д.)
2.2 Запрещается изменение комплектации этажей, мебели, их назначения, строительство дополнительных перегородок 
(искл. согласования с ЦОВЗД)
2.3 Запрещается взаимодествие с открытыми источниками огня 
(искл. Специально отведенные места: котел с лавой возле автопечки, коптильный зал)
2.4 Запрещается препятствовать проведению технического обслуживания высотки
2.5 При проведении комплексного ремонта этажа с перекрытием, запрещяется пользование перекрытым этажем до его официального открытия

## 3. Статус резидента
3.1 Резидентом признается лицо имеющее апартаменты в "Vermeer Building"
(прим. В настоящее время резедентами являются: never, 4ELOVEK, vinchiklenin)
3.2 Резиденты имеют право пользоваться общими складами без ограничения
3.3 Резиденты обладают правом частной собственности на свои апартаменты
3.4 Резидент не может проникать в чужие апартаменты и/или брать вещи без согласия на то собственника, кому принадлежат апартаменты
3.5 Резидент имеет право определять планировку своего помещения, назначение комнат, внутренний архитектурный стиль
3.6 Резидент обязан соблюдать правила пользования апартаментами, закрепленные в гл. 6 настоящего Устава ТСЖ

## 4. Статус гостя
4.1 Гостем признается лицо, не являющееся резидентом, но находящееся в границах "Vermeer Building"
4.2 Гостю разрешается посещать открытые этажи, склады
4.3 Гостю запрещается входить в апартаменты резидентов, без согласия самих резидентов
4.4 Гость не может взаимодействовать с личными складами резидентов в апартаментах, без соглсия самих резидентов

## 5. Склады
5.1 Складом является специально отведенное место для хранения ресурсов
5.2 Перечень складов устанавливается "Положением о складах ТСЖ"
